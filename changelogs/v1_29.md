I've been using Midjourney since last year to generate missing arts and tokens.

the module license.

Also I'll no longer try for every import to cover every missing arts and tokens,
especially for odds creature because it takes a lot of time, and most of the time
these are covered by the premium token pack. I would prefer to use that time covering
more import.
